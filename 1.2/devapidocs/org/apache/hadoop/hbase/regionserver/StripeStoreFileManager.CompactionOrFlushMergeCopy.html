    StripeStoreFileManager

    private class StripeStoreFileManager.CompactionOrFlushMergeCopy
extends Object
    Non-static helper class for merging compaction or flush results. Since we want to merge them atomically (more or less), it operates on lazy copies, then creates a new state object and puts it in place.
