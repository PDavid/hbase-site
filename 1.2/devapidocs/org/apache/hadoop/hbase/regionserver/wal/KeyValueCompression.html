Class KeyValueCompression

  • java.lang.Object
    • org.apache.hadoop.hbase.regionserver.wal.KeyValueCompression
  • Deprecated.

    @Deprecated
@InterfaceAudience.Private
 class KeyValueCompression
extends Object
    DO NOT USE. This class is deprecated and should only be used in pre-PB WAL. Compression class for KeyValues written to the WAL. This is not synchronized, so synchronization should be handled outside. Class only compresses and uncompresses row keys, family names, and the qualifier. More may be added depending on use patterns.
