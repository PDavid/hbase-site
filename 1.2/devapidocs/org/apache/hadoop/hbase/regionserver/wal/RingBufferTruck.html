org.apache.hadoop.hbase.regionserver.wal

Class RingBufferTruck

  • java.lang.Object
    • org.apache.hadoop.hbase.regionserver.wal.RingBufferTruck

  • @InterfaceAudience.Private
 class RingBufferTruck
extends Object
    A 'truck' to carry a payload across the FSHLog ring buffer from Handler to WAL. Has EITHER a FSWALEntry for making an append OR it has a SyncFuture to represent a 'sync' invocation. Truck instances are reused by the disruptor when it gets around to it so their payload references must be discarded on consumption to release them to GC.
